## 视频学习

### Claude Code 源码泄漏系列（抖音视频）
核心观点：
- CLI 是 AI 的母语（文本输入/结构化输出/--help 自解释）
- MCP 协议缺点：预注入上下文，Token 消耗大
- CLI-Anything：任何软件可被 Agent 驱动（HKUDS 出品）
- OpenCLI：将网站/Electron 应用转 CLI

关键发现：
- Skills 调用率问题：描述结构比代码质量重要（LangChain 评测：9% vs 82%）
- 建议：Skills ≤12 个、明确写在启动文件里、用 XML 标签分段测试
