## 本地使用流程（暑假在内地）

### 电脑翻墙

1. 打开 V2rayN（桌面文件夹 → v2rayN.exe）
2. 右键服务器列表 → 选择 **「自动配置系统代理」**
3. 打开浏览器访问 Google

### 手机翻墙

- Android：v2rayNG，配置 HTTP 代理
  - 服务器：43.160.218.220
  - 端口：15430
- iOS：Shadowrocket（付费），同上配置

### Windows 系统代理（不装 V2rayN）

设置 → 网络和 Internet → 代理 → 手动代理服务器：
- 地址：`43.160.218.220`
- 端口：`15430`

## 故障排查

**上不了网？按顺序检查：**

1. **腾讯云控制台开机？**
   控制台 → Lighthouse → first-one → 开机

2. **gost 容器挂了？**
   ```bash
   docker ps  # 检查 gost 是否在运行
   docker restart gost  # 如果没在运行则重启
   ```

3. **SSH 能连但代理不行？**
   ```bash
   ss -tlnp | grep 15430  # 确认端口在监听
   ```

4. **SSH 也连不上？**
   - 检查腾讯云防火墙是否放行了 22022
   - 用 VNC 紧急登录腾讯云控制台检查

## 服务器维护

- **续费**：腾讯云控制台 → 费用 → 续费
- **重启**：控制台 → Lighthouse → first-one → 重启
- **gost 自启**：已设置 `--restart=always`，服务器重启后会自动运行
