Course Aims: 

This course aims to provide students with basic knowledge of becoming a data scientist. The history and origins of data science will first be introduced. Stages in a data science project including data requirements, acquisition, preparation, modelling, and deployment will then be explained. Emphasis will be placed on the ethical considerations related to the use of data. Students will also learn practical skills such as data cleansing, handling of missing data, ETL (Extract, Transform, Load) processes, integration with data from open data sources, data analysis and visualisation skills. The Python programming language, which is a popular programming language used by data scientists worldwide, will be employed as a primary tool for learning practical skills.



Course Intended Learning Outcomes (CILOs)

By the end of this course, students should be able to:

CILO 1

Explain the fundamental concepts in data science including data requirements, acquisition, and preparation;

CILO 2

Apply data analysis and visualisation skills in a data science project; and

CILO 3

Discuss ethical considerations in handling data.

Statement about Generative AI Tools use in Assessed Assignment Tasks

Given the proliferation of generative AI tools and their utility in teaching and learning, in alignment with the University’s relevant principles, the default approach is that students are empowered and guided to use generative AI tools in assessed assignment tasks. Such use must align with the University’s academic integrity guidelines for students while ensuring they are achieving the PILOs and CILOs.



The “Principles for the Use of Generative AI Tools in Teaching and Learning, and Assessment” (“Principles”) guides us to use generative AI tools for better learning outcomes.
