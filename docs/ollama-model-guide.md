## 详细任务分配

### 🟦 Phi-3 Mini (3.8B) - 编程专家

**最佳场景：**
- ✅ Python/JavaScript/C++ 代码生成
- ✅ 算法实现（排序、搜索、图算法等）
- ✅ 数据结构讲解和实现
- ✅ 代码调试和错误分析
- ✅ 代码审查和优化建议
- ✅ 技术概念解释（英文）
- ✅ 数学和逻辑问题

**使用命令：**
```bash
ai phi3
# 或
ollama-run phi3
```

**示例：**
```
> 写一个Python函数实现二分查找
> 解释什么是递归，给出例子
> 这段代码报错了：IndexError: list index out of range
> 如何优化这个函数的时间复杂度
```

---

## 智能路由规则

当你使用 `ai [你的问题]` 时，系统会自动选择模型：

```
1. 包含中文 → Qwen 2.5
2. 包含编程关键词 → Phi-3 Mini
3. 简单短问题 → Llama 3.2
4. 其他 → Phi-3 Mini (默认)
```

**编程关键词：**
- python, code, programming, algorithm
- debug, function, class, error
- implement, write, optimize

---

## 注意事项

1. **显存管理**：Qwen 2.5 需要 6-8GB 显存，如果同时运行其他 GPU 任务，可能需要用 Llama 3.2
2. **首次加载**：模型第一次运行时需要加载到显存，会比较慢，之后保持运行就很快
3. **中文任务**：尽量用 Qwen，Phi-3 的中文能力有限
4. **复杂任务**：不要用 Llama 3.2，它的能力较弱

---

## 故障排除

**问题**：模型响应很慢
**解决**：检查显存是否足够，`ai status` 查看 GPU 状态

**问题**：中文输出乱码
**解决**：使用 Qwen 2.5，不要用 Phi-3 或 Llama 3.2 处理中文

**问题**：代码生成质量差
**解决**：使用 Phi-3 Mini，它专门优化了代码能力
